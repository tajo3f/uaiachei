# 🚀 Uai, Achei na Shopee | Monitor de Ofertas

![Status do Projeto](https://img.shields.io/badge/Status-Em_Desenvolvimento-orange)
![Linguagem](https://img.shields.io/badge/Linguagem-HTML%20%7C%20CSS%20%7C%20JS-yellow)

Interface moderna e responsiva de um agregador de ofertas em tempo real, focado em curadoria de bugs de preço e promoções exclusivas da Shopee Brasil.

---

## ✨ Funcionalidades Inovadoras

O projeto conta com tecnologias de ponta para melhorar a experiência do usuário e a taxa de conversão:

-   **Algoritmo de Monitoramento (Simulado):** Interface que exibe o status do rastreador de preços.
-   **Contador de Usuários em Tempo Real:** Sistema dinâmico que exibe a atividade de outros caçadores de ofertas na plataforma.
-   **Notificações Toast Dinâmicas:** Alertas inteligentes que mostram compras recentes de usuários em diferentes cidades do Brasil (Prova Social).
-   **Efeito Tilt 3D:** Cards de produtos que interagem com o movimento do rato, criando profundidade visual.
-   **Scroll Reveal:** Animações de entrada suave conforme o utilizador navega pela página.
-   **Design Ultra-Moderno:** Dark mode nativo com tipografia *Plus Jakarta Sans* e ícones *Phosphor*.

## 🛠️ Tecnologias Utilizadas

Para este projeto, foram utilizadas as seguintes tecnologias:

1.  **HTML5**: Estrutura semântica dos elementos.
2.  **CSS3 (Variáveis & Flexbox/Grid)**: Estilização avançada, filtros de desfoque (backdrop-filter) e animações personalizadas.
3.  **JavaScript (Vanilla)**: Lógica de renderização de produtos, manipulação do DOM e triggers de animação.
4.  **Phosphor Icons**: Biblioteca de ícones moderna e leve.
5.  **Google Fonts**: Fontes otimizadas para leitura digital.
